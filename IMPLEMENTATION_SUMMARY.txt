Features Implemented:
--------------------

1. 3MF Format Export:
   - Generates standard 3D Manufacturing Format files
   - Creates separate material shapes for each color
   - Each color gets its own mesh with proper material assignment
   - Uses cube primitives for each pixel
   - Outputs proper XML structure with materials and triangles
   - Packaged as ZIP file (3MF standard)

2. OpenSCAD Masks Export:
   - Generates one black/white PNG image per color
   - Creates OpenSCAD script that loads images as heightmaps
   - Uses linear_extrude to create 3D geometry from masks
   - Applies proper color to each layer
   - Packaged as ZIP file with images and .scad file

Settings Available:
------------------
- Format: 3MF or OpenSCAD
- Pixel Height: Height of each colored pixel (default 2mm)
- Base Height: Height of base layer (default 1mm)
- Pitch: Automatically calculated from material size
